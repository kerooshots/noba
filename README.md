# Noba Cutzz — Concept Website

A responsive one-page concept website for Noba Cutzz, a Dallas–Fort Worth barber.

## Publish with GitHub Pages
1. Create a new public GitHub repository.
2. Upload `index.html`, `styles.css`, and `script.js` to the repository root.
3. Go to Settings → Pages.
4. Select “Deploy from a branch,” choose `main` and `/ (root)`, then save.

## Before commercial launch
- Obtain the barber's written approval.
- Replace abstract portfolio panels with owner-approved original haircut photos.
- Confirm service descriptions, location details and booking links.
- Connect a custom domain if purchased.
